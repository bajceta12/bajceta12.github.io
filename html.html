HTML Section

HTML stand for HyperText Markup Language. Every Webpage is written using HTML! ( including this one 🐍 )

HTML is used for structuring a Webpage, while CSS and JavaScript are used to provide style and functionality. Each Webpage consists of HTML elements that themselves consist of opening tag, inner content and closing tag: example: <h1>Title</h1>. In this section, I will add html tags I learn, their usage and default styling. That way I can learn through practice, and at the end, have my HTML CheatSheet. This page will be left with no style on purpose, to show raw HTML look.
