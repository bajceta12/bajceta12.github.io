HTML Section

HTML stand for HyperText Markup Language. Every Webpage is written using HTML! ( including this one 🐍 )

HTML is used for structuring a Webpage, while CSS and JavaScript are used to provide style and functionality. Each Webpage consists of HTML elements that themselves consist of opening tag, inner content and closing tag: example: <h1>Title</h1>. In this section, I will add html tags I learn, their usage and default styling. That way I can learn through practice, and at the end, have my HTML CheatSheet.

FORM
<form></form> used to create an HTML form for user input
action attribute defines the action to be performed when the form is submitted
Usually, the form data is sent to a file on the server when the user clicks on the submit button.

target attribute specifies where to display the response that is received after submitting the form
can have one of the following values: _blank, _self, _parent, _ top, framename of iframe element

method attribute specifies the HTTP method to be used when submitting the form data
The form-data can be sent as URL variables (with method="get") or as HTTP post transaction (with method="post").



<input> most used form element, can be displayed in many ways, depending on the type attribute

<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (for selecting one of many choices)
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button 		Text

<label></label> defines a label for many form elements
element is useful for screen-reader users, because the screen-reader will read out loud the label when the user focus on the input element.
element also help users who have difficulty clicking on very small regions
<label> tag should be equal to the id attribute of the <input> element
